# Tic Tac Toe PNG

Projek ini adalah game **Tic Tac Toe** berbasis web yang menggunakan gambar PNG untuk X dan O.  
Dibuat oleh **Fakhryvx**.

## Fitur
- Tampilan modern dan responsif
- Menggunakan gambar PNG untuk X dan O
- Indikator giliran pemain (X atau O)
- Notifikasi hasil (menang/seri) langsung di halaman, bukan pop-up
- Tombol reset untuk memulai ulang permainan

## Cara Menjalankan

1. **Clone atau download** repository/folder ini.
2. Pastikan file berikut ada di dalam folder:
    - `index.html`
    - `style.css`
    - `script.js`
    - `x.png`
    - `o.png`
3. Buka file `index.html` di browser favorit Anda.

## Struktur Folder

```
e:\tictac toe\
│   index.html
│   style.css
│   script.js
│   x.png
│   o.png
│   README.md
```

## Cara Bermain

1. Klik pada kotak kosong untuk mengisi dengan X atau O sesuai giliran.
2. Permainan akan otomatis mendeteksi pemenang atau seri.
3. Klik tombol **Reset** untuk memulai permainan baru.
